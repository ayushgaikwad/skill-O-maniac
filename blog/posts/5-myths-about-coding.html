5 Myths About Coding and Developers

Saturday, Feb 12, 2022


1. CODING IS FOR GENIUSES

You might have heard from people that coding requires a high IQ or you need to be really good at math. Honestly, it’s just a myth. If you have a good logic and if you can practice anything constantly, you can become a really good developer or coder. There are some fields of course where you need to be good at math and science and all that stuff, but those are fields like game development and machine learning. You don’t need a high IQ or Einstein level math skill in fields like software or web development or to START to code.

2. YOU NEED A HIGH END SETUP

Honestly, you don’t. You can start coding even in a basic computer or a laptop. I use a Dell Latitude E6420 with Intel Core i3 and 6 GB RAM with NO SSD. I use Linux for coding and Windows for gaming and music production (dual-boot). You don’t need a high end setup to START coding or for fields like software and web development. But if we talk about game development or machine learning, then a high end setup is required.

3. DEVELOPERS CODE FOR THE ENTIRE DAY

We don’t. Let’s say for example, software development. It’s not a profession where you just code. Software developers also design softwares. You also have to explain to your client how a particular software works. You have to do presentations often. You don’t just code, you get to do many things around a single product. And yes, we do have a life!

4. YOU NEED A DEGREE

You don’t really need a degree to become a coder or to get a job in a good company. You can get hired by a good company even if you have a degree in a field which is not related to computers or even if you don’t have a degree. But on one condition that YOU HAVE EXCELLENT SKILLS. No company would hire you even if you have a master’s degree but you don’t have good skills. So, WORK ON YOUR SKILLS AND PRACTICE REGULARLY. Also, try to work on some projects or create your own projects to gain experience.

5. A.I. WILL KILL DEVELOPMENT AND CODING

No doubt that future is no-code. But not in a few decades at least. We always overestimate technology. eg. In 1900s, people said we’ll have flying cars in 2020. Tech jobs are going to be there no matter what. And their demand is increasing day-by-day because there are so many startups emerging and everyone needs either an app, software or a website. A.I. isn’t so advanced RIGHT NOW that it will replace humans.
